React Vazio
===========

Starter kit para React baseado no [Create React App](https://github.com/facebook/create-react-app) (16.13.0) com:

* Principais tags HTML 5 já componentizadas
* [Styled Components](https://github.com/styled-components/styled-components) com Mobile First e [Google Font](https://fonts.google.com/)
* [Flex Ready](https://yoksel.github.io/flex-cheatsheet/#justify-content) com Sticky Footer
* [Ant Design](https://ant.design/components/)
* [Bootstrap](https://getbootstrap.com/) via [React Bootstrap](https://react-bootstrap.github.io/)
* [React Animations](https://github.com/FormidableLabs/react-animations)
* [React Icons](https://github.com/react-icons/react-icons) (exemplo [FontAwesome](https://fontawesome.com/))
* [Axios](https://github.com/axios/axios) e exemplo usando [API REST Placeholder](https://jsonplaceholder.typicode.com)
* [Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript) via [ESlint](https://github.com/eslint)
* [Prettier](https://github.com/prettier/prettier) para padronização do código
* [React Router DOM](https://github.com/ReactTraining/react-router/tree/master/packages/react-router-dom) para navegação
